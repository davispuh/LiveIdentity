# LiveIdentity

Library utilizing IDCRL (Identity Client Runtime Library) for Microsoft Windows Live ID authentication.

## Installation

Add this line to your application's Gemfile:

    gem 'LiveIdentity'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install LiveIdentity
